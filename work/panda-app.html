Panda App

Panda App was created for budding developers, whether on a path of self-study or as part of a group or bootcamp. The app provides curated resources, job search and preparation guides, and a code portfolio page to use as a starting point.

User Flows & Wireframes

Our team dedicated one fourth of our alloted 3-week time period to developing user flows and wireframing the application. After defining application features, I had each team member walk me through their thinking process in a hypothetical app. This helped establish an agreed-upon user flow. I drew up wireframes, whose iteration led to a tab-based interface. These wires helped keep us on track in both design and devlopment.

Curated Content for Easier Learning

Panda's main aim is to provide curated content in the form of videos, PDFs, free books and other online resources to developers. Whether our users are in the early stages of their learning process or embarking upon a career in application development, Pand helps make the process easier by eliminating the time spent searching and finding valuable resources and combining them in one place.

Social Login

We decided to use Github Login for Panda. Our users are devlopers at any stage of their leraning process so it was a safe assumption that all had Github accounts. Aside from being a good and easy login/signup process, social login also gave our app the ability to access Github repos, essential for users to create a shareable profile.

Public APIs

I accessed several job board APIs (Dice, The Muse, and Indeed) to let users search for available jobs directly through our app. I also included the Meetup API so users can just as easily search for relevant groups in the area. All are filtered by a curated technology list. Github's API was also put to use to display users' code repositories.

A Professional Profile for Developers

Panda users are encouraged to easily create a public user profile. By choosing up to 4 of their favorite code repos, users can display them in a simple user profile page. I used a simple edit-in-place Javascript library to allow users to add a blurb about themselves and each code repo they chose. Becasue Panda is using their Github profile, users can choose to have their Github profile settings take precedence over Panda's, eliminating the need to re-type or copy + paste their profile info.
