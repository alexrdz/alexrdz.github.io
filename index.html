Mixtagon

About

Mixtagon is a personal project site created using Processwire CMF, an open source PHP framework and Content Management System. The Mixtagon allows members to create playlists by uploading music files and a small narrative. The purpose is to create music playlists based on a theme. Members can challenge each other on a particular theme and compete with their playlists. Playlist audio players are created with audiojs.

PHP/MySQL, Processwire CMF, CSS, jQuery

  • Role

    Developer

    • PHP developer
    • UI/UX and front-end development
    • Javascript developer (audiojs configuration)

  • Design & Development

    I created a diagram of how I wanted the application to work, and how I saw the user flow best laid out. Because I didn't have many user roles or screens to design, I skipped wireframes and I designed the entire app in the browser as I created the application itself.

Panda App

    • About

      Panda App was created for budding developers, whether on a path of self-study or as part of a group or bootcamp. The app provides curated resources, job search and preparation guides, and a code portfolio page to use as a starting point.

      View the App / View the Code

      GulpJS, Node + ExpressJS, MySQL, Sequelize ORM, SASS, Materialize CSS Framework, HandlebarsJS, Heroku

      • Role

        Technical Lead

        • Initial set-up of Express Server and configuration
        • Working with external APIs (Meetup, Dice, The Muse, and Indeed) to provide content
        • Github repository manager
        • UI/UX and front-end development (SASS & Handlebars)
        • support MySQL database integration

      • User Flows & Wireframes

        Our team dedicated one fourth of our alloted 3-week time period to developing user flows and wireframing the application. After defining application features, I had each team member walk me through their thinking process in a hypothetical app. This helped establish an agreed-upon user flow. I drew up wireframes, whose iteration led to a tab-based interface. These wires helped keep us on track in both design and devlopment.

Koala App

  • About

    Koala is a web app built to help web developers in their job-seeking process. I worked with a team of 3 talented individuals on this full-stack MERN application (MongoDB, ExpressJS, ReactJS, and NodeJS).

    Role

    Technical Lead

    • Initial set-up of Express Server and configuration
    • Working with external APIs (Meetup, Dice, The Muse, and Indeed) to provide content
    • Github repository manager
    • UI/UX and front-end development (SASS & Handlebars)
    • support MySQL database integration

    User Flows & Wireframes

    I led the team in a few exercises to determine user flows. We identified key areas where on which to focus and where pain points might arise. Because of the time limit (3 week project) we couldn't do extensive user research. We conducted a couple of interviews to inform Koala's workflow process. I wireframed the app, conducted quick user testing, and proceeded to design and development.
