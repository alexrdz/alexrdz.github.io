Mixtagon

View a Sample Playlist View the Code

  • About

    Mixtagon is a personal project site created using Processwire CMF, an open source PHP framework and Content Management System. The Mixtagon allows members to create playlists by uploading music files and a small narrative. The purpose is to create music playlists based on a theme. Members can challenge each other on a particular theme and compete with their playlists. Playlist audio players are created with audiojs.

    PHP/MySQL, Processwire CMF, CSS, jQuery

    • Role

      Developer

      • PHP developer
      • UI/UX and front-end development
      • Javascript developer (audiojs configuration)

    • Design & Development

      I created a diagram of how I wanted the application to work, and how I saw the user flow best laid out. Because I didn't have many user roles or screens to design, I skipped wireframes and I designed the entire app in the browser as I created the application itself.

  • A Fun Project and a Great CMS

    I created Mixtagon 3 years ago and it was a really fun project. I learned quite a bit of PHP programming, Object Oriented programming in PHP, and Javascript/AJAX. Processwire makes learning and writing PHP a beautiful experieince. I can't praise it enough. The only drawback I had with Processwire and PHP is that authentication and security are baked-in, so i didn't learn much in that respect.

    Leveraging Processwire's Admin

    Processwire's API is so well-written that I was able to easily display its admin to authenticated users without the need to visit the admin area. A bit of Javascript and Processwire's API helped bring the admin to the user without leaving the site or page refreshing. This allowed me to contentrate on the business logic that powered the application.

    Audio Player

    Mixtagon uses audiojs to create audio players for each playlist. It's a fantastic JS library. It also uses GetID3, a PHP library "that extracts useful information from MP3s." Users can then upload a batch of MP3s and the Mixtagon displays each file's metada including the song title, artist name, and the album. I also included text field backups in case no meta data is available. Users can also upload a companion image and add optional overlays to create a monochromatic look with a custom background.
